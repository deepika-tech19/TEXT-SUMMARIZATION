Text Summarization using NLP

This project implements an Automatic Text Summarization system using Natural Language Processing (NLP) techniques. The model processes news articles and generates concise summaries while preserving the most important information from the original text.

Features

- Text preprocessing and cleaning
- Automatic extractive text summarization
- Summary generation using NLP algorithms
- Original vs Summary comparison
- Data visualization with bar charts and word clouds
- Compression ratio analysis

Technologies Used

- Python
- Pandas
- NLTK
- Sumy
- Matplotlib
- WordCloud

Dataset

The project uses a news article dataset containing article descriptions and associated tags. The article descriptions are summarized to produce shorter and more informative versions of the original content.

Workflow

1. Load and preprocess the dataset
2. Tokenize text using NLTK
3. Apply extractive summarization using LSA Summarizer
4. Generate summaries
5. Visualize results and compare text lengths
6. Evaluate summary compression

Outcome

The system successfully reduces the length of news articles while retaining key information, making content easier and faster to understand.
